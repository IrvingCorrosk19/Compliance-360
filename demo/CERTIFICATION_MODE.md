# CERTIFICATION_MODE

## Objetivo
Validar que un usuario entiende funcionalmente el uso de Compliance 360 por rol.

## ¿Qué incluye?

- Checklist por rol (habilidades operativas mínimas).
- Evaluación final (quiz de 3 preguntas).
- Resultado con score y estado:
  - `🏅 Certificado Funcional`
  - `📘 En Progreso`
- Progreso persistente por rol en `localStorage`.

## Reglas de aprobación

Para aprobar:

1. Checklist del rol completado.
2. Quiz >= 70%.

## Roles cubiertos en certificación

- Quality Manager
- Document Controller
- Auditor
- CAPA Manager
- Risk Manager

## Flujo recomendado de uso

1. Entrenamiento por rol.
2. Práctica de pantallas.
3. Certificación (checklist + evaluación).
4. Reentrenamiento si queda en progreso.
