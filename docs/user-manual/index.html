Flujo certificado

Preparación → Revisión técnica → Aprobación interna para sometimiento → Sometimiento → Revisión de autoridad → Observación → Respuesta → Resometimiento → Decisión externa → CT/RS → Vigencia → Renovación

Nunca confunda:

  • Aprobado internamente para sometimiento = autorización interna (Approver).
  • Sometimiento registrado = envío a la autoridad (Submitter).
  • Aprobación registrada de MINSA/CSS (externa) + CT/RS activo = decisión de autoridad (Manager/QM).
