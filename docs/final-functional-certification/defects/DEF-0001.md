## Preconditions
Operating license create API available; Excel contains company constitution / ops start columns.

## Steps
1. Inspect `OperatingLicense` entity and `CreateOperatingLicenseRequest`.
2. Create license via API with only CompanyName/LicenseType/dates of license.
3. Attempt to store constitution / operations-start date from REGUTRACK.

## Expected
Dedicated fields (or Company metadata) cover Excel columns without returning to workbook.

## Actual
Only `CompanyName` / `CompanyId` / license issue-expiry dates exist. Constitution and ops-start are absent (`REGULATORY_COVERAGE_MATRIX` §C Pendiente).

## Evidence
- `src/Compliance360.Domain/RegulatoryAffairs/RegulatoryAffairsModels.Part3.cs`
- Coverage matrix Pendiente row

## Root cause
Parity gap vs REGUTRACK license sheet company header fields.

## Fix
Add `CompanyConstitutedOn` + `OperationsStartedOn` (nullable) to entity, DTO, request, UI license form, migration; import mapper.

## Regression risk
License create/list DTOs change — bump API contracts; update import.

## Retest
LIC create/update with both dates; import LICENCIAS OP row; UI display.
